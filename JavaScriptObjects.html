JavaScript array of objects from a form

.serializeArray() method, a JQuery method, creates an array of JS objects from the information provided in a form. When a user inputs data in the fields, the data is collected by the .serializeArray() into an JS object. My intention was that this JavaScript object would then be available to populate information into the portion of the app that would inform users about already scheduled events in their area.
