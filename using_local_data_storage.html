Local Storage

Local storage is a way to store data on the user's own computer rather than on a server. This is good for several reasons: first, it can be used to store data that will be available the next time the user wants it and makes loading time quicker. Second, the data is available even when the user is not online and will be available the next time the program needs it whether the user is online or has closed the browser.
